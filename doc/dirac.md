# The *dirac* prefix

The prefix **dirac** (= 12) is defined so that the units of the [Gravitic System](gravitic.pdf) can be expressed conveniently in the Harmonic System.

## Gravitic System units in Harmonic System terms

| quantity | Gravitic System unit ≈ |
|---|---|
| length | 12<sup>−4×8 + 1</sup> harmon |
| time | 12<sup>−5×8 + 1</sup> nic |
| mass | 12<sup>−1×8 + 1</sup> looloh |
| thermodynamic temperature | 12<sup>+4×8 + 1</sup> harmonic Kelvin |
| electric charge | 12<sup>−2×8 + 1</sup> universal Coulomb |

Every exponent is of the form 8*n* + 1. Since one step of the atomic / cosmic scale prefixes is 12<sup>±8</sup>, each of these units is left with a common residual factor of 12<sup>1</sup>. Extracting that factor as a prefix in its own right gives:

| quantity | with the *dirac* prefix |
|---|---|
| length | tetra-atomic dirac harmon |
| time | penta-atomic dirac nic |
| mass | atomic dirac looloh |
| thermodynamic temperature | tetra-cosmic dirac Kelvin |
| electric charge | di-atomic dirac Coulomb |

The prefix is named after Dirac — a physicist closely associated with these matters, and whose initial is 'D'.

## On the approximations

The relations above are approximate, not exact. In dozenal notation, for the length unit:

1 gravitic meter = 10;0016 <sub>4−</sub>h

([conversion](http://hosi.org/cgi-bin/conv.cgi?m=01080&d=0&fq=1.0000000&frq=10&fe=0&fu=4&tu=0&fp=0&tp=-32&fm=off&tm=off&fr=10&tr=12&fl=0&tl=0&c=Convert))

— that is, the residual factor is 10;0016 rather than exactly 10; (= 12).

## The gravitic meter and the Planck circumference

The length unit of the Gravitic System, the gravitic meter, is 35<sup>1/2</sup> times the Planck length. Loosely speaking, it is therefore a length one might call a *Planck circumference*.<sup>[1]</sup>

---

[1] Planck circumference ≈ 1.0620522 gravitic meter (since 2π / 35<sup>1/2</sup> = 1.0620522…).
